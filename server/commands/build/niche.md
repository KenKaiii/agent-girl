# /niche - Set Niche Context

Set the industry niche for optimized section generation.

## Usage
```
/niche [industry]
```

## Available Niches

| Niche | Colors | Effects | Trust Elements |
|-------|--------|---------|----------------|
| healthcare | Blue/Green | Minimal | Certifications, HIPAA |
| fintech | Navy/Gold | Glassmorphism | Security badges, PCI-DSS |
| ecommerce | Brand | Soft shadows | Reviews, guarantees |
| creative | Gradients | Full animations | Portfolio, awards |
| saas | Purple/Blue | Modern glass | Stats, integrations |
| education | Blue/Orange | Soft shadows | Accreditations |
| realestate | Navy/Gold | Elegant | Agent profiles |
| restaurant | Warm tones | Minimal | Reviews, photos |

## Task

1. **Set the niche context** for the current session
2. **Pre-research** the niche:
   - Quick web search for "{niche} website design trends 2025"
   - Note common patterns, trust elements, compliance needs
3. **Summarize** the niche preset that will be applied:
   - Color scheme
   - Effect style (minimal, glassmorphism, etc.)
   - Trust elements to include
   - Compliance requirements (HIPAA, PCI-DSS, GDPR)
   - Animation intensity (subtle, moderate, expressive)
4. **Confirm** the niche is set for subsequent `/section` commands

## Examples
```
/niche healthcare
/niche saas
/niche ecommerce
```
