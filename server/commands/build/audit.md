# /audit - Website Quality Audit

Run a comprehensive quality audit on a website or project.

## Audit Categories

1. **Performance** - PageSpeed, Core Web Vitals
2. **SEO** - Meta tags, structured data, sitemap
3. **Local SEO** - NAP, LocalBusiness schema (if applicable)
4. **Accessibility** - WCAG 2.1 AA compliance
5. **Security** - SSL, headers, vulnerabilities
6. **Third-party** - Script impact, tracker analysis

## MCP Tools

```
mcp__website2astro__verify_full → Complete 100+ checks
mcp__website2astro__audit_seo → SEO-specific audit
mcp__website2astro__audit_local_seo → Local business audit
mcp__website2astro__analyze_third_party → Script analysis
```

## Quick Check

For fast validation:
```
mcp__website2astro__verify_quick → 8-point smoke test
```

## Target Scores

- PageSpeed: 90+
- SEO: 100%
- Accessibility: WCAG 2.1 AA
- Security: A+ headers

Run the appropriate audit tools and report findings.
