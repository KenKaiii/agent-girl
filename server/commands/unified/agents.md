# Agent Pool

Verwalte den Multi-Agent Pool.

## Befehle
- `/agents list` - Alle Agents anzeigen
- `/agents status` - Pool Status
- `/agents route <task>` - Task an Agent routen

## Agent Typen
- Researcher - Recherche und Analyse
- Coder - Code-Entwicklung
- Analyst - Datenanalyse
- Optimizer - Optimierung
- Coordinator - Koordination
