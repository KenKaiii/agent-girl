# DATEV Export

Exportiere Buchungsdaten im DATEV-Format für deinen Steuerberater.

## Unterstützte Formate
- DATEV-Format (ASCII)
- Buchungsstapel
- Kontenbeschriftungen
- Mandantenstammdaten

## Buchungsfelder
- Umsatz
- Soll/Haben-Kennzeichen
- Kontonummer
- Gegenkonto
- Buchungstext
- Belegdatum
- Belegnummer

## Nutzung
Gib den Zeitraum für den Export an.
