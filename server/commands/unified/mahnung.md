# Zahlungserinnerung / Mahnung

Erstelle eine §286 BGB konforme Zahlungserinnerung.

## Mahnstufen
1. **Zahlungserinnerung** - Freundliche Erinnerung
2. **1. Mahnung** - Förmliche Mahnung mit Fristsetzung
3. **2. Mahnung** - Letzte Mahnung vor rechtlichen Schritten
4. **Mahnbescheid** - Gerichtliches Mahnverfahren

## Rechtliche Grundlagen
- §286 BGB - Verzug des Schuldners
- §288 BGB - Verzugszinsen (5% über Basiszinssatz)
- §288 Abs. 5 BGB - Pauschale 40€ bei B2B

## Nutzung
Gib Rechnungsnummer und Kundendaten an.
