# Kassenbuch

GoBD-konforme Kassenbuchführung.

## Pflichtangaben
- Datum
- Fortlaufende Belegnummer
- Buchungstext
- Einnahme oder Ausgabe
- Kassensaldo

## GoBD Anforderungen
- Unveränderbarkeit der Einträge
- Vollständigkeit
- Zeitnahe Erfassung
- Ordnung und Übersichtlichkeit

## Nutzung
Gib die Kassenbewegung ein (Datum, Betrag, Beschreibung).
