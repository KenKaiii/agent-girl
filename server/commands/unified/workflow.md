# Workflow Management

Verwalte DAG-basierte Workflows.

## Befehle
- `/workflow list` - Alle Workflows anzeigen
- `/workflow start <id>` - Workflow starten
- `/workflow status <id>` - Status prüfen
- `/workflow stop <id>` - Workflow stoppen

## Workflow Features
- Checkpoints für Wiederaufnahme
- Parallele Ausführung
- Fehlerbehandlung
- Scheduled Execution
