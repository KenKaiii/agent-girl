## SMART SECTION GENERATION

When creating sections, ALWAYS:

### 1. Pre-Research (if niche specified)
- Quick web search for "{niche} website design trends 2025"
- Note: color schemes, trust elements, compliance needs

### 2. Generate Unique IDs
- NEVER use predictable patterns (section-1, hero-section, main-content)
- Vary pattern per section using these formats:
  - Semantic: `s_a7x3k2` (prefix + hash)
  - Narrative: `swift-wave-3k` (adjective-noun-hash)
  - Pure hash: `_x7k2m9p` (underscore + hash)
  - Mixed: `sec-3a-bloom` (prefix-hash-noun)
  - Data: `v7x9k2` (v + hash)
  - Element: `el_calm_2x` (el + adjective + hash)

### 3. Apply Modern Effects
- Soft shadows: `shadow-[0_4px_20px_rgba(0,0,0,0.08)]`
- Smooth hover: `hover:-translate-y-1 transition-all duration-200`
- Subtle gradients: `bg-gradient-to-br from-blue-50 to-purple-50`
- Glassmorphism: `backdrop-blur-md bg-white/60 border border-white/20`
- Neumorphism: `shadow-[8px_8px_16px_#d1d9e6,-8px_-8px_16px_#ffffff]`
