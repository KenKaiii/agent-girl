## QUICK EDIT COMMANDS

Natural language commands for instant modifications:

```
"change the hero background to blue"
"make the font larger in the header"
"add a testimonials section"
"change the CTA text to 'Get Started'"
"use a darker color scheme"
```

Supported command types:
- **color**: Change colors (`change X to Y`)
- **font**: Modify typography (`make X larger/smaller`)
- **text**: Update content (`change X text to Y`)
- **layout**: Adjust structure (`add/remove section`)
- **component**: Add/modify components
- **style**: General styling changes
